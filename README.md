# Twin Trees Fayetteville

Modern restaurant website for Twin Trees Fayetteville - Original Syracuse Style pizza since 1957.

**Live:** https://twin-trees-fayetteville.vercel.app

## Pages
- `/` - Homepage (hero, reviews, menu preview, gallery, location, contact)
- `/menu` - Full menu with prices
- `/about` - Our Story
- `/location` - Hours, directions, map
- `/contact` - Contact form, phone, socials, catering
- `/gallery` - Photo gallery
- `/order` - Order online landing page

## Stack
- Next.js 14 + TypeScript + Tailwind CSS
- Deployed on Vercel
- Fonts: Playfair Display + Lato (Google Fonts)

## Development
```bash
npm install
npm run dev     # localhost:3000
npm run build   # production build
```
